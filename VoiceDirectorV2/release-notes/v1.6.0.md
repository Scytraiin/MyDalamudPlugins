# Voice Director v1.6.0

## Summary

Baseline Voice Director release before the API 15 workspace modernization.

## Highlights

- Supports changing the cutscene voice language on a per-duty basis.
- Restores the configured default language when no override applies.
- Ships with custom repository metadata for Dalamud distribution.

## Notes

- This release note is kept as the starting point for future release preparation.
- API 15 migration and containerized validation were added after this baseline.
