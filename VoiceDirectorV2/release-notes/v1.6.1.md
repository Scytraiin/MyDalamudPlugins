# Voice Director v1.6.1

## Summary

Dalamud API 15 compatibility and tooling refresh release for Voice Director, bringing the plugin, validation flow, and release packaging in line with the newer SDK and the local Dalamud 15 runtime bundle.

## Highlights

- Migrated the plugin project to `Dalamud.NET.Sdk/15.0.0` and `net10.0-windows`.
- Replaced the older static plugin-service pattern with constructor-based service injection and safer UI error handling.
- Refined the configuration window flow for default-language selection, duty override management, and override removal.
- Added a dedicated `VoiceDirector.Tests` project covering voice-selection behavior and repository metadata alignment.
- Added Docker-based validation that runs tests first, then builds the plugin against a mounted Dalamud dev folder.
- Added a release preparation script and release artifact workflow for `out/release/latest.zip` and `out/release/scyt.repo.json`.

## Notes

- This release is intended for the newer Dalamud API 15 baseline and was validated against the local `LootInfoPlugin/15.0.0.2/dev` runtime bundle in this workspace.
- In-game smoke testing is still recommended after publishing because the plugin changes a live game config value during territory transitions.
