# Loot History v2.0.1

## Summary
Follow-up release for the API 15 branch that makes Need/Greed/Pass information more visible across the main plugin UI.

## Highlights
- Added a dedicated rolls summary column to compact mode so roll data is visible without opening the full browser.
- Added a rolls column to the Item Details table, with a matching visibility toggle in plugin settings.
- Expanded the Overview tab with rolled-entry counts, total roll-event counts, and recipient-focused buckets.
- Enriched overview buckets so item, zone, and recipient summaries can show roll activity alongside entry counts.
- Extended overview summary tests to cover the new roll-aware aggregates.

## Notes
- The UI work is validated against the local Dalamud `15.0.0.2/dev` bundle.
- Roll visibility is improved here, but actual in-game roll capture still depends on the incoming message/payload format being matched successfully.
